# 博客文章大纲生成器

## 模板

```
我想让你扮演一位专业的内容策划专家。请为以下主题创建一个详细的博客文章大纲：

主题：[你的主题]
目标读者：[目标人群]
文章目的：[教育/娱乐/说服/等]
字数要求：[X] 字

请生成一个包含以下部分的大纲：

1. 标题选项（3个备选）
   - 悬念型
   - 数字型  
   - 利益型

2. 导语（50字以内）
   - 制造痛点/好奇心

3. 正文结构（至少5个主要章节）
   对每个章节：
   - 章节标题
   - 核心观点（1句）
   - 支撑论据/案例（2-3个）
   - 章节字数估算

4. 结论部分
   - 总结要点
   - Call to Action

5. SEO优化建议
   - 主关键词
   - 5个长尾关键词
   - Meta描述（150字以内）

请用中文输出。
```

## 使用说明

1. 将 [ ] 中的内容替换为你的具体信息
2. 根据需要调整章节数量
3. 使用大纲后让AI直接写初稿

## 示例输入

主题：如何用AI提升工作效率
目标读者：职场人士
文章目的：教育
字数要求：2000字

## 适用场景

- 写技术博客
- 内容营销
- 教育类文章
- 产品介绍
