# 邮件营销序列生成器

## 模板

```
请为我创建一个完整的邮件营销序列，用于：[营销目的]

产品/服务：[产品或服务描述]
目标客户：[目标人群描述]
主要利益点：[列出3-5个核心利益]

请生成以下7封邮件的序列：

## 邮件1：欢迎/介绍邮件
- 主题行（3个选项）
- 预览文本
- 正文（100字以内）
- CTA

## 邮件2：问题放大邮件
- 主题行
- 预览文本
- 痛点描述（3个场景）
- 情感共鸣段落

## 邮件3：解决方案邮件
- 主题行
- 产品/服务介绍
- 核心功能（3个）
- 社会证明

## 邮件4：演示/案例邮件
- 主题行
- 客户成功案例
- 数据/成果展示
- 链接或附件

## 邮件5：紧迫感邮件
- 主题行
- 限时优惠说明
- 倒计时提示
- 额外价值

## 邮件6：最终提醒邮件
- 主题行
- 简洁提醒
- 最后一个障碍清除
- 最终CTA

## 邮件7：最后告别邮件
- 主题行
- 温和的告别
- 保持关系的后门
- 未来通讯选项

## 发送时间建议
每封邮件之间的间隔建议

请用中文输出。
```

## 使用说明

1. 替换所有 [ ] 中的内容
2. 根据实际产品调整邮件数量
3. A/B测试不同主题行
4. 监控打开率和点击率

## 适用场景

- 产品发布
- 促销活动中
- 客户培育
- 课程销售
- 软件订阅
