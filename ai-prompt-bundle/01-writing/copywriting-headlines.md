# 广告文案标题生成器

## 模板

```
请为以下产品/服务生成10种不同风格的广告文案标题：

产品/服务：[产品或服务描述]
核心卖点：[主要卖点]
目标受众：[目标人群]
平台：[使用的平台，如小红书/朋友圈/抖音/等]

## 请生成以下风格：

1. 数字型（包含具体数字）
2. 悬念型（制造好奇心）
3. 利益型（强调好处）
4. 紧迫型（限时/限量）
5. 故事型（简短故事）
6. 对比型（前后对比）
7. 问答型（提问引发思考）
8. 情绪型（情感共鸣）
9. 权威型（专家/名人背书）
10. 简单直接型（直击要点）

## 每条标题要求

- 标题：[标题内容]
- 副标题：[补充说明，可选]
- 适用位置：[信息流/搜索广告/开屏广告等]
- 字符数：[XX字]

## 最终建议

选出最吸引人的3条标题，并说明理由。
```

## 使用说明

1. 替换 [ ] 中的具体信息
2. 根据平台特性调整风格比例
3. A/B测试不同风格
4. 跟踪CTR优化

## 适用平台

- 微信朋友圈
- 抖音/快手
- 小红书
- 百度搜索
- 微博
- Facebook/Instagram
