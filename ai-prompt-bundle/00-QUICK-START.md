# AI Prompt Templates Bundle - Quick Start Guide

So you got the bundle. Here's how to actually use it.

## What's Inside

20+ templates for common tasks. Nothing fancy, just stuff that works.

## How to Use

Pick a template that fits what you're trying to do. Paste it into ChatGPT or Claude or whatever you're using. Fill in the brackets with your situation. Hit send. That's it.

## The Categories

**Writing** - blog stuff, emails, social posts, that kind of thing
**Coding** - code reviews, bug hunts, API planning
**Marketing** - SEO articles, competitor research, product copy
**Research** - market analysis, user interviews, trend tracking
**Productivity** - meetings, task breakdowns, decisions

## A Couple Tips

Don't treat these like magic formulas. They're starting points. Adjust them to match how you actually talk and what you're actually trying to accomplish.

If something doesn't feel right when you use it, probably means you need to tweak it. That's normal.
