# 🚀 AI Prompt Templates Bundle - 快速开始指南

## 📦 包含内容

**50+ 专业级提示词模板**，覆盖5大核心场景

## ⚡ 快速使用

### 第一步：选择模板

根据你的需求，选择对应分类的模板：

| 需求 | 使用模板 |
|-----|---------|
| 写博客/文章 | `01-writing/blog-outline.md` |
| 写营销文案 | `01-writing/copywriting-headlines.md` |
| 写社交媒体 | `01-writing/social-media-post.md` |
| 开发代码 | `02-coding/code-review.md` |
| 找Bug | `02-coding/bug-fix-template.md` |
| 做SEO | `03-marketing/seo-article.md` |
| 做竞调 | `03-marketing/competitor-analysis.md` |
| 做市场研究 | `04-research/market-analysis.md` |
| 整理会议 | `05-productivity/meeting-notes.md` |
| 做决策 | `05-productivity/decision-framework.md` |

### 第二步：复制模板

打开对应的 `.md` 文件，复制全部内容。

### 第三步：填入信息

将 `[你的主题]`、`[目标人群]` 等占位符替换为你的具体信息。

### 第四步：发送给AI

将填写好的模板发送给 ChatGPT、Claude、Gemini 或任何 AI 工具。

### 第五步：获得结果

AI 会根据模板生成专业级的内容。

## 💡 使用技巧

### 如何获得最佳效果

1. **具体化你的输入** — 越具体，AI输出越精准
2. **保留模板结构** — 结构化输入是高质量输出的关键
3. **迭代优化** — 根据输出调整你的输入
4. **组合使用** — 多个模板可以组合使用

### 模板定制化

每个模板都可以根据你的需求调整：
- 删减不需要的部分
- 增加额外的约束
- 调整输出格式
- 添加行业特定的要求
