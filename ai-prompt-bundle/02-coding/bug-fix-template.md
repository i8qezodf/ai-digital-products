# Bug修复助手提示词

## 模板

```
## Bug报告

**问题描述：**
[详细描述bug的表现]

**期望行为：**
[用户期望的结果]

**实际行为：**
[当前实际的结果]

**复现步骤：**
1. [步骤1]
2. [步骤2]
3. [步骤3]

**环境信息：**
- 操作系统：[如 Windows 11]
- 编程语言版本：[如 Python 3.11]
- 相关库/框架：[库名和版本]
- 服务器/客户端：[相关信息]

**错误日志：**
```
[粘贴错误日志]
```

**相关代码：**
```[语言]
[粘贴相关代码片段]
```

## 请分析

1. **可能原因**（列出最可能的3个）
2. **推荐修复方案**（给出具体代码）
3. **预防措施**（如何避免类似问题）
4. **测试建议**（如何验证修复）
```

## 使用说明

1. 填写所有 [ ] 中的信息
2. 尽可能提供完整的错误日志
3. 只贴相关代码，不要贴整个项目
4. 按推荐顺序检查可能原因

## 示例

**问题描述：**
用户登录后点击"我的订单"页面空白

**期望行为：**
显示用户的订单列表

**实际行为：**
页面显示空白，无任何错误提示

**复现步骤：**
1. 打开APP
2. 登录账号
3. 点击"我的订单"
4. 页面空白
