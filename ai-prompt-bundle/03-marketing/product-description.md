# 产品描述文案生成器

## 模板

```
请为以下产品撰写营销描述：

## 产品信息

**产品名称：**
[产品名]

**产品类型：**
[软件/硬件/服务/课程/等]

**核心功能（3-5个）：**
1. [功能1]
2. [功能2]
3. [功能3]

**目标用户：**
[用户画像描述]

**主要利益：**
1. [利益1 - 尽可能量化]
2. [利益2]
3. [利益3]

**价格：**
[价格信息]

**购买/使用链接：**
[链接]

## 请生成

### 1. 短描述（20字以内）
用于APP商店、广告语

### 2. 中描述（100字以内）
用于官网、社交媒体

### 3. 长描述（300字以内）
用于应用商店、产品页面

### 4. 核心卖点（5个）
每个用一句话，格式：🎯 [卖点]

### 5. 使用场景（3个）
每个包含：场景描述 + 如何使用 + 效果

### 6. FAQ（5个）
常见问题及解答

### 7. 社会证明示例
如果有好评/数据，生成示例文案

### 8. CTA文案
- 按钮文字
- 落地页文案
```

## 使用说明

1. 填写所有产品信息
2. 中描述要包含核心关键词（SEO）
3. 长描述使用情感化语言
4. FAQ要覆盖真实疑虑

## 适用场景

- 应用商店描述
- 官网产品页
- 社交媒体帖子
- 广告落地页
- 电子书/白皮书
