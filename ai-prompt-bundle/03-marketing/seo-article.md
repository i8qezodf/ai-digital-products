# SEO文章生成器

## 模板

```
请为我撰写一篇SEO优化的文章。

## 核心关键词
主关键词：[如 "AI写作工具"]
长尾关键词：[列出5-8个长尾词]

## 文章要求
- 目标字数：[如 2000字]
- 目标读者：[如 市场营销人员]
- 难度级别：[初学者/中级/高级]
- 文章类型：[教程/评测/指南/对比/等]

## SEO要求
- 关键词密度：2-3%
- H1标题中包含主关键词
- 至少3个小标题包含关键词
- 开头100字内包含主关键词
- 结尾包含CTA

## 内容大纲（可选）
[如果你有的大纲，或者留空让AI生成]

## 参考资料
[相关链接或文档，可选]

## 请生成

1. **标题选项**（3个，包含关键词）
2. **Meta描述**（150字符以内）
3. **文章正文**
   - 引言（制造痛点）
   - 目录
   - 各章节内容（至少5个H2）
   - 常见问题（FAQ）
   - 结论
4. **内部链接建议**
5. **相关关键词**（扩展内容用）

## 格式要求

- H1：文章主标题（1个）
- H2：主要章节（3-5个）
- H3：子章节
- 段落简洁，单句不超过30字
- 使用列表和表格增强可读性
```

## 使用说明

1. 用 [ ] 标记的内容替换为你的信息
2. 关键词不要堆砌，自然融入
3. 生成后自己检查可读性
4. 添加相关图片建议

## SEO最佳实践

- 每800-1000字使用一个H2
- 图片添加alt文本
- 内部链接3-5个
- 外部权威链接2-3个
- 表格/列表增强可读性
