# 市场分析报告模板

## 模板

```
请对以下市场进行全面分析：

## 分析对象

**市场/行业：**
[如 中国SaaS市场]

**分析目的：**
[如 评估进入该市场的可行性]

**地理范围：**
[如 全球/中国/北美/等]

**时间范围：**
[如 2024-2029]

## 请分析

### 1. 市场概述

- 市场规模（金额/数量）
- 增长率及趋势
- 市场驱动因素
- 市场制约因素

### 2. 市场细分

| 细分市场 | 规模 | 增长率 | 利润率 | 吸引力 |
|---------|------|--------|-------|-------|
| 细分A | | | | |
| 细分B | | | | |

### 3. 竞争格局

- 主要玩家（TOP 5）
- 市场份额分布
- 竞争壁垒
- 新进入者威胁

### 4. 客户分析

- 目标客户画像
- 购买决策因素
- 需求痛点
- 采购周期

### 5. 趋势与机会

- 技术趋势
- 政策趋势
- 消费趋势
- 潜在机会

### 6. 风险与挑战

- 市场风险
- 竞争风险
- 技术风险
- 政策风险

### 7. 进入策略建议

- 目标细分市场
- 价值主张
- 渠道策略
- 定价策略

### 8. 数据来源

请尽量引用权威数据来源（如 Gartner, Forrester, 艾瑞, CNNIC 等）

## 输出格式

使用Markdown格式，包含：
- 思维导图式的大纲
- 数据表格
- 关键洞察高亮
- 可执行建议标记
```

## 使用说明

1. 明确分析目的和范围
2. 提供尽可能多的背景信息
3. 要求引用权威数据源
4. 重点关注中国市场的政策因素
