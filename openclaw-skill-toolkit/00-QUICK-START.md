# OpenClaw Skill Toolkit - Quick Start

You've got a collection of skills that work with OpenClaw. Here's how to get going.

## What This Is

Pre-built skill configurations for OpenClaw — the AI assistant framework. Each skill is a specialized tool for common tasks.

## How to Install

1. Copy the skill folder you want into your OpenClaw skills directory
2. Restart OpenClaw or run `openclaw reload`
3. The skill activates automatically when relevant

Location: `%USERPROFILE%\.openclaw\workspace\skills\`

- Adjust and customize. These are starting points, not scripts.
