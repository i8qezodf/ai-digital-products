# Social Post Skill

Write social media content that doesn't sound like it was written by a committee.

## When to Use

Creating content for Twitter/X, LinkedIn, or similar platforms. Use when you want posts that feel human, not corporate.

## The Skill

```
Write a social post:

PLATFORM: [Twitter/X / LinkedIn / Threads / other]
TOPIC: [what you're posting about]
GOAL: [engagement / clicks / followers / thought leadership]
ANGLE: [specific take or hook]

CONTEXT:
- Who should see this: [audience]
- What you want them to do: [engage / click / share / think]
- Your perspective on this: [opinion, experience, or just the facts]
- Examples/data available: [if any]

GENERATE:

Option 1: [Personal story / opinion take]
[1-2 sentence hook that makes someone stop scrolling]
[Main point — the thing you're saying]
[Why it matters or call to action]

Option 2: [List / thread starter]
Hook that gets attention
[Number] things about [topic]:
1. [Point 1]
2. [Point 2]
3. [Point 3]
[Closing thought / CTA]

Option 3: [Question / engagement bait]
[Provocative or genuine question]
[Context if needed]
[Who you want to answer]

FORMAT RULES:
Twitter: Under 280 chars, no threads unless requested
LinkedIn: 150-300 words works best, professional but not stiff
Keep it punchy — first sentence is everything
No hashtags unless absolutely necessary
```

## Example

```
PLATFORM: Twitter/X
TOPIC: Why most "productivity tips" don't work
ANGLE: Opinion, from experience
GOAL: Engagement and shares

OUTPUT:

I've tried every productivity system out there.
Things that actually worked: writing things down, saying no, doing hard stuff first.
Things that didn't: apps, frameworks, $500 courses.

The productivity industry sells you the problem so you buy the solution.

If you feel unproductive, it's usually not a system problem.
It's a decision problem.

What's the one thing you're avoiding that would change everything?

---

Or:

5 productivity myths that cost me years:
1. Wake up at 5am (I just got tired)
2. No time for deep work (I wasted 3 hours on email)
3. More hours = more output (diminishing returns hit fast)
4. You need a perfect system first (start messy, fix later)
5. Productivity is about doing more (it's about doing what matters)

Your mileage may vary. Mine did.
```

## Tips

- Write like you're texting a smart friend, not giving a presentation
- Controversial but genuine > safe but boring
- Specific > generic ("I cut meetings from 10 to 3 per week" > "I improved efficiency")
- Edit ruthlessly — if a word doesn't do work, cut it
- Timing matters — post when your audience is active

## Variations

**Thread:** Extend into multi-post thread with hook + substance
**Quote card:** Just the hook, for image format
**Reply:** Write as response to a topic/trend
**Thread summary:** After tweeting a thread, add single-tweet summary
