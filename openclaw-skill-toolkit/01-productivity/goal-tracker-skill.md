# Goal Tracker Skill

Track progress on goals without the motivational fluff.

## When to Use

You set a goal and want to track whether you're actually making progress. Use weekly or at key milestones.

## The Skill

```
Track progress on this goal:

GOAL: [what you're trying to achieve]
TIMEFRAME: [by when]
MEASUREMENT: [how you'll know you're done]

PAST PROGRESS:
[what happened since last check-in]

CURRENT STATUS:
- Completed: [what you finished]
- In Progress: [what you're working on right now]
- Blocked: [what's stopped]

PREDICTION:
Will you hit the goal on time?
- [ ] Yes, on track
- [ ] Probably, minor concerns
- [ ] Uncertain, need to reassess
- [ ] Behind, need to pivot

IF BEHIND:
What's the gap? [current vs. planned]
What could close it? [options]
What are you going to do? [commitment]

NEXT CHECK-IN: [when]
```

## Example

```
GOAL: 100 newsletter subscribers
TIMEFRAME: 90 days
MEASUREMENT: Count of confirmed opt-ins

PAST PROGRESS:
Week 1-4: Set up Substack, wrote 4 issues, no promotion
Week 5-8: Shared on Twitter twice, added link to email signature

CURRENT STATUS:
- Completed: Platform setup, 4 issues published, Twitter presence
- In Progress: Building Twitter following
- Blocked: None

PREDICTION: Uncertain, need to reassess

IF BEHIND:
Gap: 8 subscribers current vs. ~33 expected at this pace
Options: Guest post on someone's newsletter, Reddit promotion, collaboration
What I'll do: Guest post outreach starting next week

NEXT CHECK-IN: 2 weeks
```

## Tips

- Be honest about blockers — they don't go away by ignoring them
- If you keep getting blocked the same way, that's the real problem
- Small consistent progress beats sporadic bursts
- Measure outcomes, not just activities

## Variations

**Habit tracking:** Daily check-in format, streak counter
**Project milestones:** Break into phases, check each milestone
**Team goals:** Add "owner" field to each item
