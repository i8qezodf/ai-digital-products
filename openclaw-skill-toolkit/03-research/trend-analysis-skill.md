# Trend Analysis Skill

Track and make sense of trends affecting your space.

## When to Use

Strategic planning, content planning, product roadmap decisions. Use quarterly or when something significant shifts in your industry.

## The Skill

```
Analyze trends in:

AREA: [industry, technology, market, etc.]
TIME HORIZON: [3 months / 1 year / 3 years]
FOCUS: [specific angle or broad scan]

Process:

1. TREND IDENTIFICATION
   List the major trends you see:
   - [Trend 1]: [description], [source/signals]
   - [Trend 2]: [description], [source/signals]
   - [Trend 3]: ...

2. TREND ANALYSIS
   For each major trend:
   
   What's happening:
   - [Description of the shift]
   
   Signals that it's real:
   - [Evidence 1]
   - [Evidence 2]
   - [Data points]
   
   Who's driving it:
   - [Key players, demographics]
   
   Why it's happening:
   - [Underlying causes]
   
   Momentum:
   - [Accelerating / Stable / Slowing]
   - [Evidence for assessment]

3. IMPACT ASSESSMENT
   How does this affect [your product/market/strategy]?
   - [Immediate implications]
   - [Medium-term shifts]
   - [Long-term restructuring]

4. RESPONSE OPTIONS
   For each significant trend:
   - Do nothing (monitor)
   - Adapt (tactical changes)
   - Embrace (build it in)
   - Lead (go early)

5. CONFIDENCE & SOURCES
   - How confident are we?
   - Key sources: [list]
```

## Example

```
AREA: AI productivity tools for developers
TIME HORIZON: 1 year
FOCUS: How AI is changing developer workflows

TREND 1: AI pair programmer adoption
- Signals: GitHub Copilot 1M+ users, Cursor growth, JetBrains AI assistant
- Drivers: Productivity gains (studies show 30-50%), reduced boilerplate
- Momentum: Accelerating — new entrants weekly

TREND 2: From code generation to code review
- Signals: Code review AI tools emerging, AI debugging assistants
- Drivers: Generated code needs validation, bottleneck shifting
- Momentum: Stable — growing category

TREND 3: Local-first AI
- Signals: LM Studio, Ollama, privacy concerns
- Drivers: Data privacy, cost, latency
- Momentum: Accelerating — hardware improving fast

IMPACT:
- Developers will spend less time on boilerplate, more on design
- Code review becomes critical (AI generates fast, humans validate)
- Teams need AI policies and workflows
- Privacy-sensitive code will stay local

RESPONSE OPTIONS:
- Embrace AI code gen (integrate into roadmap)
- Lead code review AI (build or partner)
- Monitor local-first (watch for right timing)
```

## Tips

- Distinguish between trends and fads — ask "will this matter in 3 years?"
- Look at adjacent industries for leading indicators
- Talk to practitioners, not just analysts
- Trends interact — track combinations, not just individual shifts

## Variations

**Content calendar:** Focus on trends affecting content strategy
**Product roadmap:** Focus on tech trends and user behavior shifts
**Investment thesis:** Focus on market and financial trends
